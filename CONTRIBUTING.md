# Contributing to SafeText 🚀

Thanks for contributing!

## Setup

```bash
git clone <repo>
cd repo
git checkout develop
dart pub get
```

## Create Branch

```bash
git checkout -b feature/your-feature
```

## Guidelines

- Follow code style
- Add tests
- Keep PR small

## Run checks
```bash
dart analyze
dart test
```

## Submit PR
- Target branch: develop
- Ensure CI passes
